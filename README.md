# Gerador de Senhas da Lola

Projeto feito em HTML, CSS e JavaScript puro.

## Recursos

- Tamanho configurável entre 4 e 64 caracteres
- Letras maiúsculas e minúsculas
- Números e símbolos
- Opção para remover caracteres parecidos
- Geração aleatória com `crypto.getRandomValues`
- Garantia de pelo menos um caractere de cada grupo selecionado
- Cálculo estimado de entropia em bits
- Indicador visual de força
- Botão para copiar a senha
- Layout responsivo e acessível

## Como abrir

Abra o arquivo `index.html` no navegador.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie `index.html`, `style.css` e `script.js`.
3. Vá em **Settings > Pages**.
4. Em **Source**, selecione a branch `main` e a pasta `/root`.
5. Salve e aguarde o link ser criado.
